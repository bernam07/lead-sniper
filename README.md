**G-Maps Lead Extractor** is a powerful desktop automation tool designed for marketing agencies and freelancers. It automates the process of extracting business leads (Name, Phone, Website) directly from Google Maps results.

## üöÄ Key Features

* **GUI Interface:** Modern, dark-mode interface built with `CustomTkinter`. No terminal needed.
* **Deep Extraction:** Clicks on each business to retrieve hidden details like **Phone Numbers** and **Websites**.
* **Export to CSV:** Automatically saves clean data to Excel/CSV ready for cold calling or email marketing.
* **Infinite Scroll Handling:** Automatically loads more results as it scrapes.
* **Headless Mode:** Run extraction in the background without opening the browser window.
